== Installation ==
1. Login to WordPress Admin for your website
2. Click on "Plugins" in the left sidebar, then "Add New"
3. Search for "Lorem Expert Help"
4. Install and Activate the "Lorem: Expert Website Help” plugin
5. Once Activated, click the Lorem button in the bottom-right corner whenever you need help with your site
